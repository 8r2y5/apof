#APOF

## Instalation

[Install Docker](https://docs.docker.com/engine/installation/linux/ubuntulinux/)
[Install Compose](https://docs.docker.com/compose/install/)
Clone this repo
```
cd /your/dev/path/
git clone git@github.com:PyPila/apof.git
cd apof
```
Provision MySQL 
```
docker-compose up -d db
```

## Work instance
```
docker-compose up 
```
